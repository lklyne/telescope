## Scenario

Create a sticky note on the canvas with some initial text.
Verify the note appears with the correct content.
Edit the note's text to something different.
Verify the updated text is displayed.
Delete the note and confirm it is gone.

## Expected outcomes
- A sticky note appears on the canvas with the initial text
- After editing, the note displays the updated text
- After deletion, the note is no longer on the canvas

## Cleanup
- Delete any text notes created during this test
